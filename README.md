# springBoot项目
## 基于shiro的角色权限管理


### 依赖

 <dependency>
  <groupId>org.apache.shiro</groupId>
   <artifactId>shiro-spring</artifactId>
   <version>1.4.0</version>
 </dependency>

|表| user | role | permission | user_role | role_permission |
